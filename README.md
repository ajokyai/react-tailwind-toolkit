# React + TailwindCSS — Beginner's Toolkit 🚀

> Moringa AI Capstone Project

## How to Run

1. Clone the repo
   git clone https://github.com/ajokyai/react-tailwind-toolkit.git

2. Go into the folder
   cd react-tailwind-toolkit

3. Install dependencies
   npm install

4. Start the app
   npm run dev

5. Open your browser at http://localhost:5173

## What You'll See
A responsive developer profile card with a gradient banner, avatar, stats, and hover effects — built entirely with TailwindCSS utility classes.

## Tech Stack
- React 18
- TailwindCSS v3
- Vite
